环境准备与软件安装

Node.js

官网：http://nodejs.cn/

hexo

  • 安装hexo： npm install -g hexo-cli
  • 在博客目录下初始化：hexo init
  • 启动： hexo s
  • 本地打开：http://localhost:4000/

将Hexo部署到Github

发布文章

hexo new “文章名”
hexo clean
hexo generate
hexo deploy
